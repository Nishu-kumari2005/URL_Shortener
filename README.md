🔗 URL Shortener Web Application

A simple and efficient URL Shortener built using Node.js, Express, MongoDB, and EJS.
This application converts long URLs into short, shareable links and tracks the number of clicks for each shortened URL.

## Features
- Shorten long URLs
- Redirect to original URL using short link
- User login and signup
- View all created short URLs
- Click count for each URL
- Link expiration support

---

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: JWT
- Styling: CSS / Tailwind CSS
